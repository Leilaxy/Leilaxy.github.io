Technical Implementation

I used the unity engine face tracking technology , which will code models to track a person's face and overlay. Four or five of these models were set up and the multi face tracking code was used to allow them to track faces randomly. This technique is a script for multi face tracking that uses AR Face Management from the AR Foundation to detect and track faces in the scene and place a randomly selected AR face model on each detected face.

For the models I chose 0 and 1 to represent the models. Mostly these two models are relied upon to produce various effects. Using random generation techniques to clone the 0 and 1 models at random within a certain range, the script can loop through multiple object models and adjust them to random positions and rotations within a set maximum distance. Other adjustments can be made to these codes to produce different effects, for example to limit the random cloning to the xy plane to cover the billboard model.
